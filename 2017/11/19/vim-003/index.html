《vim从入门到放弃》003　编辑之道

发表于 2017-11-19

我发现，如果你要说服一个人做某件事情，基本上是不可能的。只有忽悠加欺骗，才有可能达成目标。无论你怎样摆事实，讲道理，都不如唱首歌来得直接。这里面的原因可能是说的不如唱的好听吧。

无论是挖黄河的时候，挖出一个只有一只眼的石头人，并且配上元朝的民谣，“莫道石人一只眼，挑动黄河天下反”。还是“骑白马，挎洋枪。三哥哥吃了八路军的粮，有心回家看姑娘，呼儿嗨哟，打日本也顾不上。三八枪，没盖盖。八路军当兵的没太太。待到那打下榆林城，呼儿嗨哟，一人一个女学生！”

所以，几乎所有说服别人用vim的努力，其实都是徒劳的。当别人说到骑白马的时候，首先想到的是榆林城里的一个女学生；当别人说到乐视的时候，首先想到的是为梦想窒息；当别人说到锤子的时候，首先想到的是每家都有的一种常用来砸钉子的工具；当别人说到小米的时候，首先想到的是为发烧而生；当别人说到vim的时候，首先想到的是vim是啥？

讲真，vim界缺少一个有影响力的大忽悠！

听约炮界的达人说，虚与委蛇要比开门见山好用的多。他们说，想卖手机，你得谈情怀；想卖会员，你得谈梦想；想上床，你得讲故事。我觉得好有道理，那天周日，我真的讲了一晚上软件的故事，讲到口干舌燥，东方渐白，就直接从咖啡馆出门骑了个单车去上班了。

我打算讲点编辑的道理，然后旁敲侧击的说vim做这些事情是最好的，因为明说vim是世界上最好的编辑器，这个是违反广告法的。我把这篇文章叫编辑的艺术，或者编辑之道，哪个题目能忽悠住人，我以后有时间做个AB测试。下面，正文开始了。

我觉得中文比英文在某些方面要有些优点。最近，twitter网站把每条tweet的140字符的限制，从140字符，放宽到280个字符。同志们，280个字啊，3条tweet就是一篇高考作文啊！这是什么概念！汉字的信息密度比英文的信息密度的不知道哪里去了…

比如”How old are you?”对应的中文”怎么老是你？”，英文是12个字符，中文是5个字符。如果要删除的话，英文要按12次删除键，中文要按5次删键。当然，有个小技巧是按住删除键不动，在刚删完的瞬间抬起来，否则就删多了。当然还有个方法是，用鼠标选定以后，再删。一手键盘，一手鼠标，高手啊！

如果是删3个单词呢？删2个句子呢？删2个段落呢？当然只能是一手鼠标选，一手键盘删了，高手都是这么弄的。毕竟按20秒删除键实在是有点太不专业了！

如果能把单词，句子，段落，文章当成一个整体的对象就好了。当然了，现在稍微专业点的编辑器都支持这种操作。可以一次编辑一个单词，一个句子，一个段落。更有意思的是，根本用不着鼠标选定，双手根本不用离开键盘就能搞定这一切。

因为中文操作的词，英文操作的是字母，所以，中文操作的层次会更高一点。字母　<　词语　<　句子　<　段落　<　文章　<　秘书。你越是从更高层次上操作文本，编辑起来就越有效率。最高的层次的文本编辑是找个秘书写稿子。

再想想，你最喜欢的编辑器如何做到如下的操作：

  1. 让光标向下移动10行
  2. 把光标下面这个单词删了，外加单词后面的空格也不要了
  3. 把这个文件第50行到100行中的所有的typo替换成type
  4. 光标下面的单词是controller.rb，我知道这是个文件，我想快速打开它
  5. 现在5:59分了，还有1分钟就下班了，工作没做完，想保存所有的环境，回家加班 :)
  6. 我经常要打“软件那些事儿”这个词，我希望打rjn就自动的展开
  7. 一个gfwlist文件有2000行，我只要每一行前面的域名部分，如何快速的去除其它部分
  8. 一个文件，都是小写字母的，我想把每一句的首字母改成大写的
  9. 复制粘贴了好几个部分，要重复使用，如何方便的切换粘贴板中的内容？总不能每次都重复的复制一遍吧？
  10. 同时打开一个文件在两个窗口，查看并编辑不同的部分，方便我抄代码

你最喜欢的编辑器如何快速的做到这些呢？我知道，左手键盘右手鼠标，肯定是没问题的。vim做以上的事情，可能就是几个按键就搞定了。因为宣传vim是最好的编辑器是违反广告法的，国家规定，在广告中不能用最好这个词。我觉得，vim是上半球和下半球比较好用的编辑器之一。
